தமிழ் எழுத்துகள் மொத்தம் 247.

1. உயிர் எழுத்துகள் = 12
2. மெய் எழுத்துகள் = 18
3. உயிர்மெய் எழுத்துகள் = 12 x 18 = 247
4. ஆயுத எழுத்த = 1
  
தமிழ் எழுத்துகள் மொத்தம் = 12 + 18 + (12 x 18) + 1
